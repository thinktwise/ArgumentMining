Argument Annotated Essays Corpus (AAEC) (version 1.0)
-----------------------------------------------------

The corpus contains 90 persuasive essays annotated with argument components and
argumentative relations. The folder contains the following files:

1. The annotation guidelines used in this study (guidelines.pdf)

2. A CSV-File containing the source URL and the topic of each essay (topics.csv)
  
3. The brat-project including the essays and the annotations (annotations.zip)


Explore annotations using brat
------------------------------
   
For annotating the data, we used the brat annotation tool (version 1.3 "Crunchy Frog"), 
which can be downloaded from http://brat.nlplab.org 

For exploring the annotation using brat, copy the content of the annotation.zip file to
/bratDir/data/AAEC
